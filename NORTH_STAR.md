# CodeCanvas North Star

## 한 줄 목표

CodeCanvas의 최종목표는 코드를 읽지 않아도 코드베이스의 로직을 이해하고 리뷰하고 질문할 수 있게 만드는 것이다.

## 우리가 만들고 있는 것

CodeCanvas는 그래프 도구가 아니다.
CodeCanvas는 코드베이스를 사람이 읽을 수 있는 `로직 모델`로 바꾸는 시스템이다.

사용자는 파일 트리나 함수 목록이 아니라 아래를 봐야 한다.

- 이 요청은 어디로 들어오는가
- 어떤 인증, 분기, 서비스, DB, 외부 API를 거치는가
- 왜 이 응답이 나왔는가
- 어디가 위험한가
- 무엇을 먼저 리뷰해야 하는가

## North Star 경험

가장 이상적인 경험은 이렇다.

1. 사용자가 엔트리포인트를 고른다.
2. CodeCanvas가 실제 실행 경로와 가능한 경로를 분리해서 보여준다.
3. 줌아웃하면 전체 요청 흐름이 설명된다.
4. 줌인하면 파일, 함수, 분기, assignment, return, side effect까지 내려간다.
5. 사용자는 코드 원문을 거의 열지 않고도 1차 리뷰를 끝낸다.

## 제품의 핵심 약속

### 1. Zoom out = explain

축소했을 때는 구조 요약이 아니라 로직 설명이 나와야 한다.

사용자는 그래프를 보고 이런 문장을 말할 수 있어야 한다.

- 인증 후 세션을 찾는다
- 세션이 없으면 404를 반환한다
- 있으면 supervisor가 응답을 생성한다
- 위험 지점은 권한 체크와 외부 호출이다

### 2. Zoom in = inspect

확대했을 때는 추상적 설명이 아니라 실제 구현 구조가 드러나야 한다.

- Level 0: 요청 설명
- Level 1: 레이어 설명
- Level 2: 파일 / 클래스
- Level 3: 함수 / 메서드
- Level 4: 함수 내부 로직
- Level 5: CFG와 실제 실행된 분기 경로

### 3. Runtime is primary, static is supporting

기본 화면은 항상 실제로 탄 경로를 중심으로 해야 한다.
정적 분석은 드릴다운과 가능한 경로 설명에 사용한다.

즉 사용자는 항상 아래 3가지를 구분해서 봐야 한다.

- 실제 실행된 것
- 정적으로 가능한 것
- 추론했지만 아직 검증되지 않은 것

### 4. Review is the product

우리가 파는 것은 예쁜 시각화가 아니라 리뷰 속도와 이해 속도다.

그래프 위에서 바로 보여줘야 하는 신호는 이런 것들이다.

- 인증 / 권한 체크
- 예외 경로
- DB write
- 외부 API 호출
- 불확실한 연결
- 변경 영향 범위
- 테스트 커버리지 공백

## 기술적으로 도달하고 싶은 끝점

장기적으로 CodeCanvas는 아래를 할 수 있어야 한다.

- 요청 하나를 넣으면 응답까지의 전체 로직을 설명한다
- 함수 내부에서 어떤 assignment / branch / return이 결과를 만들었는지 보여준다
- 완전한 정적 CFG를 만들고 nested branch를 포함한 가능한 경로를 표현한다
- runtime trace를 CFG 위에 겹쳐서 실제 탄 경로를 정확히 표시한다
- PR diff를 보고 어떤 요청 흐름이 영향을 받는지 보여준다
- 리스크를 점수화하고 코드 리뷰 우선순위를 제안한다
- 사용자의 질문에 대해 그래프와 근거를 바탕으로 답한다

## 우리가 절대 잊으면 안 되는 것

Python은 동적이다.
그래서 CodeCanvas의 목표는 100% 완전한 추론기가 아니라, 근거와 불확실성을 분리해서 보여주는 고신뢰 로직 설명기다.

이 원칙은 항상 유지해야 한다.

- 확정 사실을 추정처럼 흐리지 않는다
- 추정을 확정처럼 포장하지 않는다
- 런타임 검증 결과를 가장 높은 신뢰도로 둔다
- 사용자가 언제든 근거 파일과 라인으로 내려갈 수 있게 한다

## 성공의 기준

CodeCanvas가 성공했다는 것은 사용자가 이렇게 말하는 상태다.

- 코드를 거의 안 읽었는데도 흐름을 설명할 수 있다
- 어디가 위험한지 바로 보인다
- 어디를 먼저 리뷰해야 할지 안다
- 왜 이 응답이 나왔는지 말할 수 있다
- 필요할 때만 코드 원문을 열면 된다

## 한 문장 결론

CodeCanvas는 코드 시각화 툴이 아니라, 코드베이스를 리뷰 가능한 로직으로 번역하는 실행 설명기다.
